# Heimdall v0.9.0-beta.1 — Release Notes

**Release date:** 2026-04-27  
**Type:** Beta pre-release  
**Tag:** `v0.9.0-beta.1`

> **This is a beta release.** APIs, configuration format, and wire behaviour may
> still change before v1.0.  The beta is suitable for staging environments and
> advanced users who can tolerate breaking changes.  Do not deploy in hostile
> network environments until the external audit sign-off is published.
> Please report issues at https://github.com/FlavioCFOliveira/Heimdall/issues.

---

## Minimum Supported Rust Version (MSRV)

**Rust 1.94.0** — required for Rust Edition 2024 features and `let`-chain stabilisation
(RFC 2497).  The workspace is pinned to the nightly channel via `rust-toolchain.toml`
until Rust 1.94 is released as stable.

---

## Upgrade Path from v0.9.0-alpha.1

This beta release requires no configuration changes relative to alpha.  Drop-in
upgrade:

```sh
# Verify the binary (cosign keyless)
cosign verify-blob \
  --bundle heimdall-v0.9.0-beta.1-x86_64-unknown-linux-musl.cosign.bundle \
  heimdall-v0.9.0-beta.1-x86_64-unknown-linux-musl

# Replace binary and restart
systemctl stop heimdall
install -m 755 heimdall-v0.9.0-beta.1-x86_64-unknown-linux-musl /usr/sbin/heimdall
systemctl start heimdall
```

No data-migration steps are required.  The cache and NTA stores are forward-compatible.

## Known Limitations

- **External security audit**: not yet complete.  Do not deploy in hostile network
  environments until the Sprint 41 audit sign-off is published.
- **SLSA provenance**: hash binding is a stub (ENG-080).  Provenance attestations
  are generated but hash binding is incomplete.
- **cargo install**: `--check-config` flag not yet implemented.
- **riscv64**: uses glibc (not musl) until the riscv64-musl target stabilises.
- **macOS**: development target only; sandbox profile not enforced automatically.

## Artefact Verification

All release binaries are signed with cosign (keyless OIDC, Sigstore transparency log).
Verify a downloaded binary:

```sh
cosign verify-blob \
  --bundle heimdall-v0.9.0-beta.1-x86_64-unknown-linux-musl.cosign.bundle \
  heimdall-v0.9.0-beta.1-x86_64-unknown-linux-musl
```

A signed CycloneDX SBOM (`sbom-v0.9.0-beta.1.cdx.json`) is attached to this release.
