# Heimdall v1.0.0-rc.1 — Release Notes

**Release date:** 2026-04-27  
**Type:** Release candidate  
**Tag:** `v1.0.0-rc.1`

> **This is a release candidate.** APIs and configuration format are frozen.
> Wire behaviour is stable.  Please test this release in staging environments
> and report any issues tagged `rc-blocker` at
> https://github.com/FlavioCFOliveira/Heimdall/issues.
> GA will be tagged when the RC-blocker milestone reaches zero and the
> external audit sign-off is published.

## Minimum Supported Rust Version (MSRV)

**Rust 1.94.0** — required for Rust Edition 2024 features and `let`-chain stabilisation
(RFC 2497).  The workspace is pinned to the nightly channel via `rust-toolchain.toml`
until Rust 1.94 is released as stable.

---

## Upgrade Path

### From v0.9.0-beta.1

No configuration changes required.  Drop-in upgrade:

```sh
cosign verify-blob \
  --bundle heimdall-v1.0.0-rc.1-x86_64-unknown-linux-musl.cosign.bundle \
  heimdall-v1.0.0-rc.1-x86_64-unknown-linux-musl

systemctl stop heimdall
install -m 755 heimdall-v1.0.0-rc.1-x86_64-unknown-linux-musl /usr/sbin/heimdall
systemctl start heimdall
```

### From v0.9.0-alpha.1

Follow the beta.1 upgrade path first, then this path.  No multi-hop data
migration is required; the cache and NTA stores are forward-compatible across
the entire 0.9.0-alpha.1 → 1.0.0-rc.1 range.

a breaking-change GA bump (v2.0.0).

## Known Limitations

- **External security audit**: sign-off not yet published. GA is blocked on this.
- **SLSA provenance**: hash binding is a stub (ENG-080).
- **cargo install**: `--check-config` flag not yet implemented.
- **riscv64**: uses glibc (not musl) until the riscv64-musl target stabilises.
- **macOS**: development target only; sandbox profile not enforced automatically.

## GA Promotion Criteria

GA (`v1.0.0`) will be tagged when all of the following are true:

1. RC-blocker milestone open count = 0.
2. External audit sign-off (`docs/audit/sign-off-template.md`) published and
   linked from the GA release.
3. No Critical or High findings from the audit remain open.
4. Tier 3 nightly CI green at the tag commit.
